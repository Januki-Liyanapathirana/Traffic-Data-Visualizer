#  Traffic Data Visualizer and Validator

This project is a Python-based application that validates user-input dates, processes traffic data from CSV files, and visualizes the data using a graphical interface built with Tkinter.

---

##  Key Features

- **Date Validation**
  - Ensures users enter a valid date in the `DD-MM-YYYY` format  
  - Handles invalid inputs with clear and user-friendly error messages  

- **CSV Data Processing**
  - Reads and processes traffic data from CSV files  
  - Extracts relevant insights based on the selected date  

- **Data Visualization**
  - Displays graphical representations of traffic flow  
  - Focuses on two junctions for easy comparison and understanding  

- **User Interface**
  - Built using Tkinter for an interactive and simple GUI experience  

---

## 🛠️ Technologies Used

- Python  
- Tkinter (for GUI)  
- CSV (for data handling)  

---

##  How It Works

1. User enters a date in the required format  
2. The system validates the input  
3. Traffic data is loaded from CSV files  
4. Relevant data is extracted and visualized graphically  

---

##  Purpose

This project helps in understanding traffic patterns by combining data validation, processing, and visualization into a single user-friendly application.
